# HRC HUB - Super Admin Web Portal

This is the Super Admin Web Application for the HoReCa Hub platform. It is a completely independent React application built with Vite.

## Roles Supported
- Super Admin (No regular admin role)

## Setup and Running

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the local development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
